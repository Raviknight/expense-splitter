## Build note (IMPORTANT for future deploys)
- build.mjs PROD emits a content-hashed bundle (bundle-<hash>.js) and rewrites docs/index.html.
  DEV (`npm run dev`) emits a fixed docs/bundle.js. Running the dev server CLOBBERS the prod
  docs/ build. So ALWAYS run `npm run build` immediately before `git add/commit/push` to deploy.
